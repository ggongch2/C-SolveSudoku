# C-SolveSudoku

C++로 스도쿠를 푸는 알고리즘입니다.
예시)
0 7 0 0 2 0 8 0 0 
0 0 0 0 0 8 0 0 7
8 6 0 0 4 0 0 2 3
0 0 0 2 0 0 5 0 0
0 0 3 7 0 9 6 0 0
0 0 6 0 0 4 0 0 0
7 1 0 0 3 0 0 6 9
9 0 0 4 0 0 0 0 0
0 0 4 0 9 0 0 7 0

이런식으로 입력을 하면 

-----------------------------------------------------------------
4 7 9 3 2 1 8 5 6 
3 2 1 6 5 8 9 4 7 
8 6 5 9 4 7 1 2 3 
1 8 7 2 6 3 5 9 4 
5 4 3 7 1 9 6 8 2 
2 9 6 5 8 4 7 3 1 
7 1 2 8 3 5 4 6 9 
9 3 8 4 7 6 2 1 5 
6 5 4 1 9 2 3 7 8 
-----------------------------------------------------------------
0 7 0 0 2 0 8 0 0
0 0 0 0 0 8 0 0 7
8 6 0 0 4 0 0 2 3
0 0 0 2 0 0 5 0 0
0 0 3 7 0 9 6 0 0
0 0 6 0 0 4 0 0 0
7 1 0 0 3 0 0 6 9
9 0 0 4 0 0 0 0 0
0 0 4 0 9 0 0 7 0

이런형식으로 답과 원래 문제를 출력해줍니다.

빈칸 의 경우는 0으로 채우면 되며, 나머지는 한칸씩 띄워 그대로 채우시면 됩니다.
